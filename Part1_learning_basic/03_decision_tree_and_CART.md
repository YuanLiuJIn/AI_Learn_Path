# 03 决策树与 CART

决策树是一种非常直观的模型。它像一套判断题：每次根据一个特征做一次分裂，最后走到叶子节点得到预测结果。

## 1. 决策树的直觉

假设你要判断一个人是否会买某款游戏：

```text
是否玩过同类型游戏？
  是 -> 最近 7 天是否活跃？
      是 -> 可能购买
      否 -> 不一定购买
  否 -> 是否看过广告？
      是 -> 可能尝试
      否 -> 不太可能购买
```

这就是一棵树。

每个内部节点是一个问题，每条边是答案，每个叶子节点是预测结果。

## 2. 决策树能做什么

- 分类：预测类别，例如是否违约、是否流失。
- 回归：预测数值，例如房价、销量。

分类树的叶子节点通常输出类别或类别概率。

回归树的叶子节点通常输出该叶子中样本标签的平均值。

## 3. 树如何选择分裂点

核心问题：

> 当前节点应该用哪个特征、哪个阈值来分裂，才能让子节点更“纯”？

“纯”的意思是：同一个节点里的样本尽量属于同一类，或者数值尽量接近。

## 4. 分类树常用分裂标准

### 4.1 熵 Entropy

熵衡量混乱程度。

- 如果一个节点全是正类，熵很低。
- 如果正负各一半，熵很高。

信息增益表示分裂后熵减少了多少。

```text
信息增益 = 分裂前熵 - 分裂后加权平均熵
```

ID3、C4.5 常用信息增益或信息增益率。

### 4.2 基尼指数 Gini

CART 分类树常用基尼指数。

二分类时：

```text
Gini = 1 - p1^2 - p0^2
```

直觉：随机抽两个样本，它们类别不同的概率。Gini 越小，节点越纯。

## 5. 回归树常用分裂标准

回归树通常希望分裂后每个子节点内部的数值更接近。

常用标准是均方误差或方差减少：

```text
选择让左右子节点 MSE 加权和最小的分裂
```

叶子节点预测值通常是该叶子中训练样本标签的平均值。

## 6. 什么是 CART

CART 是 Classification and Regression Trees 的缩写，来自 Breiman 等人在 1984 年出版的经典教材 *Classification and Regression Trees*。

CART 的关键特点：

- 既能做分类，也能做回归。
- 每次都是二叉分裂。
- 分类常用 Gini。
- 回归常用平方误差。
- 支持剪枝。

很多现代树模型和集成模型都以 CART 为基础。

## 7. CART 分类树示例

假设有一个二分类任务：判断用户是否会购买。

特征：

- 年龄
- 最近 7 天登录次数
- 是否看过广告

CART 会尝试很多候选分裂：

```text
年龄 <= 25?
年龄 <= 30?
登录次数 <= 3?
登录次数 <= 10?
是否看过广告 == 是?
```

它会选择让 Gini 降低最多的那个分裂。

## 8. CART 回归树示例

预测房价时，CART 可能先问：

```text
面积 <= 90 平米？
```

如果是，再继续问：

```text
是否靠近地铁？
```

最终每个叶子节点输出一个平均房价。

这不是拟合一条连续曲线，而是把空间切成很多块，每块给一个平均值。

## 9. 决策树为什么容易过拟合

如果不限制树的深度，决策树可以一直分裂到每个叶子只有很少样本。这样训练集效果很好，但泛化能力差。

常见限制：

- `max_depth`：最大深度。
- `min_samples_split`：节点至少多少样本才允许继续分裂。
- `min_samples_leaf`：叶子节点最少样本数。
- `max_leaf_nodes`：最大叶子节点数。

## 10. 剪枝

剪枝是减少过拟合的重要方法。

### 10.1 预剪枝

在树生长过程中提前停止。

例如限制最大深度、最小叶子样本数。

### 10.2 后剪枝

先长出一棵较大的树，再删除对验证集帮助不大的分支。

CART 经典方法使用代价复杂度剪枝。

## 11. 决策树的优缺点

优点：

- 易解释。
- 能处理非线性关系。
- 对特征缩放不敏感。
- 可处理数值和类别特征。
- 能捕捉特征交互。

缺点：

- 容易过拟合。
- 对数据扰动敏感。
- 单棵树预测能力有限。
- 贪心分裂不保证全局最优。

## 12. 决策树与线性模型对比

| 维度 | 线性模型 | 决策树 |
|---|---|---|
| 决策边界 | 线性 | 阶梯状、非线性 |
| 可解释性 | 参数可解释 | 路径可解释 |
| 特征缩放 | 通常需要 | 通常不需要 |
| 过拟合风险 | 中等 | 高 |
| 表达特征交互 | 需要手工构造 | 可自动捕捉 |

## 13. 与随机森林和 GBDT 的关系

随机森林和 GBDT 都是“很多棵树”的模型。

- 随机森林：很多棵树并行训练，最后投票或平均。
- GBDT：很多棵树按顺序训练，后一棵树修正前面模型的错误。

理解 CART 是理解这些模型的前提。

## 本章小结

决策树用一系列问题把样本空间切分成不同区域。CART 是现代树模型的重要基础，使用二叉分裂，既能分类也能回归。单棵树容易过拟合，因此后续需要随机森林和 GBDT 这类集成方法。
