# 02 线性模型与优化基础

线性模型是机器学习中最重要的基础模型之一。它简单、可解释、训练快，也是理解神经网络的入口。

## 1. 为什么先学线性模型

线性模型看起来简单，但它包含了机器学习的核心思想：

```text
预测 = 参数 × 特征
损失 = 预测与真实答案的差距
训练 = 调整参数让损失变小
```

神经网络可以粗略看作很多层线性变换加非线性激活的组合。

## 2. 线性回归

线性回归用于预测连续值。

### 2.1 一维形式

```text
y_hat = w * x + b
```

含义：

- `x`：输入特征，例如房屋面积。
- `w`：权重，表示特征影响有多大。
- `b`：偏置，表示基础值。
- `y_hat`：预测值。

例如房价预测：

```text
预测房价 = 价格系数 × 面积 + 基础价格
```

### 2.2 多维形式

多个特征时：

```text
y_hat = w1*x1 + w2*x2 + ... + wn*xn + b
```

矩阵写法：

```text
y_hat = Xw + b
```

这就是后续张量计算的基础。

## 3. 损失函数：MSE

线性回归常用均方误差：

```text
MSE = mean((y_hat - y)^2)
```

为什么要平方？

- 正负误差不会抵消。
- 大错误会被放大。
- 数学上容易求导。

如果异常值很多，MAE 可能更稳健。

## 4. 最小二乘法与梯度下降

线性回归有两种常见求解方式。

### 4.1 最小二乘法

直接求出让 MSE 最小的参数。优点是精确，缺点是数据很大或特征很多时计算成本高。

### 4.2 梯度下降

梯度下降不直接求答案，而是一步步调整参数：

```text
参数 = 参数 - 学习率 × 梯度
```

直觉：

- 损失函数像一座山。
- 当前参数是山上的一个点。
- 梯度告诉你上坡最快方向。
- 要让损失下降，就往梯度相反方向走。

## 5. 学习率

学习率决定每次走多大一步。

| 学习率 | 结果 |
|---|---|
| 太小 | 收敛很慢 |
| 合适 | 稳定下降 |
| 太大 | 来回震荡，甚至发散 |

深度学习训练中，学习率常常比模型结构更影响结果。

## 6. 逻辑回归

逻辑回归名字里有“回归”，但通常用于分类。

它先做线性计算：

```text
z = w*x + b
```

然后用 sigmoid 把结果压到 0 到 1：

```text
p = sigmoid(z) = 1 / (1 + exp(-z))
```

`p` 可以理解为属于正类的概率。

例如：

- `p = 0.9`：很可能是正类。
- `p = 0.1`：很可能是负类。

## 7. 交叉熵损失

二分类常用二元交叉熵：

```text
loss = -[y*log(p) + (1-y)*log(1-p)]
```

直觉：

- 真实是 1，但模型预测 0.99，损失很小。
- 真实是 1，但模型预测 0.01，损失很大。

大语言模型训练中的 next token prediction 也常用交叉熵，只不过是多分类版本。

## 8. 正则化

正则化用于控制模型复杂度，减少过拟合。

### 8.1 L2 正则化

在损失中加入权重平方：

```text
loss = 原始损失 + λ * sum(w^2)
```

效果：让权重不要太大。

L2 正则化也叫 Ridge Regression。在深度学习中，类似思想常通过 weight decay 实现。

### 8.2 L1 正则化

在损失中加入权重绝对值：

```text
loss = 原始损失 + λ * sum(|w|)
```

效果：让部分权重变成 0，有特征选择作用。

L1 正则化也叫 Lasso Regression。

## 9. 特征缩放

线性模型和梯度下降通常需要特征缩放。

如果一个特征范围是 0 到 1，另一个是 0 到 100000，梯度下降会变得困难。

常用方法：

- 标准化：减均值除以标准差。
- 归一化：缩放到 0 到 1。

树模型一般不太需要特征缩放。

## 10. 线性模型的优缺点

优点：

- 简单
- 快速
- 可解释
- 适合做 baseline
- 在高维稀疏数据上效果好

缺点：

- 表达能力有限
- 难以自动学习复杂非线性关系
- 需要较好的特征工程

## 11. 与决策树、GBDT 的对比

| 模型 | 适合场景 | 特点 |
|---|---|---|
| 线性回归 | 关系接近线性、需要解释性 | 简单稳定 |
| 决策树 | 非线性、规则明显 | 容易过拟合 |
| 随机森林 | 表格数据、降低过拟合 | 稳定但解释稍弱 |
| GBDT | 表格数据竞赛、强 baseline | 效果强，调参重要 |

## 12. 与深度学习的关系

一个神经网络层本质上通常是：

```text
线性变换 -> 非线性激活
```

例如：

```text
h = ReLU(Wx + b)
```

如果没有非线性激活，多层线性网络仍然等价于一层线性模型。因此非线性是神经网络表达复杂函数的关键。

## 本章小结

线性模型是理解机器学习训练过程的最短路径：输入特征，计算预测，定义损失，通过优化算法更新参数。后面的神经网络只是把这个过程扩大到更多参数、更复杂结构和更大数据。
