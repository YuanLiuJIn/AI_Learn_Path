# 00 学习路线（网络结构篇）

## 总体目标

理解从 CNN/RNN 到 Transformer 的演化逻辑，能讲清每种结构“解决了上一代的什么问题”，并能写出最小可运行实现。

## 一条主线：每种结构都在解决前一种的痛点

| 结构 | 解决的核心问题 | 引入的关键思想 |
|---|---|---|
| MLP | 基础非线性拟合 | 全连接 + 激活 |
| CNN | 图像参数太多、需要平移不变性 | 局部连接、权重共享、卷积 |
| RNN | 处理变长序列、利用时间顺序 | 隐藏状态在时间步间传递 |
| LSTM/GRU | RNN 长程依赖遗忘、梯度消失 | 门控机制控制信息流 |
| Seq2Seq | 输入输出长度不一致（如翻译） | 编码器-解码器 + 上下文向量 |
| Attention | 单一上下文向量信息瓶颈 | 解码每步动态对齐输入 |
| ResNet | 网络太深无法训练 | 残差连接 `x + F(x)` |
| Transformer | RNN 无法并行、长程依赖弱 | 自注意力 + 位置编码，完全抛弃循环 |

## 建议学习节奏（约 8~10 周）

### 第 1~2 周：CNN

- 卷积、padding、stride、池化、感受野
- 经典网络：LeNet、AlexNet、VGG
- 产出：手写一个 CNN 跑 MNIST/CIFAR-10

### 第 3 周：RNN

- 隐藏状态、时间步展开、BPTT
- 梯度消失/爆炸为什么在 RNN 里更严重
- 产出：能画出 RNN 按时间展开的计算图

### 第 4 周：LSTM 与 GRU

- 遗忘门、输入门、输出门
- GRU 的简化
- 产出：能解释门控如何“记住”长期信息

### 第 5 周：RNN 语言模型

- 字符级/词级语言模型
- 采样生成、温度
- 产出：手写字符级 RNN，能生成文本

### 第 6 周：Seq2Seq

- 编码器-解码器
- teacher forcing
- 产出：手写一个玩具翻译模型（如数字翻译、日期格式转换）

### 第 7 周：Attention

- 对齐问题、QKV、注意力权重
- 产出：能手推一次注意力计算

### 第 8 周：RNN + Attention

- 把注意力接到 Seq2Seq 解码器
- 可视化注意力对齐矩阵
- 产出：带注意力的翻译模型，画出对齐热力图

### 第 9 周：ResNet

- 退化问题、残差块、恒等映射
- 产出：实现 ResidualBlock，对比深网络有无残差的训练曲线

### 第 10 周：Transformer

- 自注意力、多头、位置编码、FFN、残差+LayerNorm
- Encoder/Decoder、掩码
- 产出：实现一个小型 Transformer block，理解每个张量 shape

## 学习原则

- 每个结构先问“它解决了什么问题”，再看公式。
- 每章都要落到一份能跑的最小代码，哪怕用玩具数据。
- 重点理解张量 shape 的变化，这是看懂结构的关键。
- 经典论文读摘要和图即可，不必一开始抠所有细节。
