# AI_Learn

这是一个用于系统学习大模型相关知识的长期学习项目。当前先完成第一部分：`Part1_learning_basic`，目标是把机器学习、树模型、集成学习、深度学习基础、GPU/CUDA 与 PyTorch 串成一条可执行的学习路线。

## 项目阶段

- `Part1_learning_basic/`：基础能力建设，已创建。
- 后续计划：网络结构、内容生成、语言模型、强化学习等。

## 使用方式

1. 先读 `Part1_learning_basic/README.md`，了解学习顺序。
2. 按章节阅读，每章做最少一个小练习。
3. 遇到公式不要只背结论，要能解释“为什么需要这个公式”，必要时查 `09_math_foundations_for_ml.md`。
4. 每学完一个主题，用 `08_practice_projects.md` 中的小项目验证理解。

## 学习原则

- 先理解问题，再学习算法。
- 先能手推小例子，再上框架和代码。
- 先掌握传统机器学习，再进入深度学习。
- 先会训练和评估模型，再追求复杂网络结构。
