# 00 学习路线（语言模型篇）

## 总体目标

理解 NLP 从“词向量”到“大语言模型”再到“多模态”的演化，能讲清每个里程碑解决了什么问题。

## 演化与解决的问题

| 里程碑 | 解决的问题 | 关键思想 |
|---|---|---|
| Word2Vec | 词怎么变成有语义的向量 | 上下文预测，词义 = 向量 |
| ELMo | 一词多义 | 上下文相关的动态词向量 |
| Transformer | RNN 无法并行、长程弱 | 自注意力（见 Part2）|
| BERT | 深度双向理解 | 掩码语言模型 MLM + 预训练微调 |
| GPT | 通用生成与少样本 | 自回归 + 规模化 + in-context learning |
| T5 | 任务格式不统一 | 一切都是“文本到文本” |
| CLIP / Flamingo | 让模型看懂图 | 图文对齐 / 视觉接入语言模型 |

## 建议节奏（约 6~8 周）

```text
第1周  10 篇论文串烧，建立全局地图
第2周  Word2Vec → ELMo → BERT 表征学习
第3周  手写一个最小 Agent（理解工具调用与局限）
第4-5周 GPT-1/2/3 进化，预训练范式、in-context learning
第6周  T5 与 encoder-decoder 统一框架
第7-8周 多模态：CLIP 对比学习、Flamingo 视觉接入
```

## 学习原则

- 抓住两条主线：表征（怎么理解）和生成（怎么产出）。
- 反复对照 BERT（理解型）vs GPT（生成型）。
- 每个模型都问：预训练任务是什么？怎么用到下游？
- 多读原始论文的图（BERT、GPT、T5、CLIP 的结构图非常清晰）。
