# 02 损失函数背后是最大似然估计（MLE）

## 1. 为什么先讲 MLE

VAE、扩散模型、自回归语言模型，甚至分类用的交叉熵，**损失函数几乎都能追溯到同一个根基：最大似然估计（Maximum Likelihood Estimation, MLE）**。理解了 MLE，你就抓住了所有生成模型损失的“母公式”。

## 2. 一句话直觉

> 最大似然就是：**调整模型参数，让模型觉得“已经发生的训练数据”最有可能发生。**

生活类比：你有一枚不知是否均匀的硬币，抛 10 次得到 7 正 3 反。你会猜“正面概率约 0.7”——因为这个猜测让“7 正 3 反”这件已发生的事看起来最合理。这就是最大似然。

## 3. 形式化

给定训练数据 `x_1, ..., x_N`，模型分布 `p_θ(x)`。我们希望参数 `θ` 让这些数据的联合概率最大：

```text
最大化  ∏ p_θ(x_i)        （各样本概率连乘）
```

连乘不好优化（数值会下溢），取对数变成连加（log 单调，不改变最优点）：

```text
最大化  Σ log p_θ(x_i)        （对数似然）
```

机器学习习惯“最小化损失”，所以取负号：

```text
最小化  -Σ log p_θ(x_i)        （负对数似然 NLL）
```

> 这就是负对数似然损失（NLL）。它是无数损失函数的统一形式。

## 4. MLE 等价于最小化 KL 散度

更深一层：最大似然 ≈ 让模型分布 `p_θ` 逼近真实分布 `p_data`。

衡量两个分布差距用 **KL 散度**：

```text
KL(p_data || p_θ) = E_{x~p_data}[ log p_data(x) - log p_θ(x) ]
```

其中 `log p_data(x)` 与 `θ` 无关，所以：

```text
最小化 KL(p_data || p_θ)  ⟺  最大化 E[log p_θ(x)]  ⟺  最大似然
```

一句话：

> 最大似然，本质就是让模型分布在 KL 意义下尽量贴近真实数据分布。

## 5. MLE 如何变成你熟悉的损失

### 5.1 分类 → 交叉熵

若 `p_θ(y|x)` 是分类概率，对它做最大似然，推出来正好是**交叉熵损失**。所以 Part1 学的交叉熵，就是 MLE 的特例。

### 5.2 回归（高斯假设）→ MSE

若假设输出服从高斯分布 `y ~ N(f_θ(x), σ²)`，对它做最大似然：

```text
-log p_θ(y|x) ∝ (y - f_θ(x))²  + 常数
```

正好是**均方误差 MSE**！所以回归用 MSE，背后是“噪声是高斯”的最大似然假设。

### 5.3 语言模型 → next token 交叉熵

GPT 训练目标“预测下一个 token”，就是对 `p_θ(token_t | token_{<t})` 做最大似然，损失是逐 token 的交叉熵。

### 5.4 生成模型的难点：p_θ(x) 算不出来

对图像这种高维数据，`p_θ(x)` 通常**无法直接计算**。于是各类生成模型用不同技巧绕开：

```text
VAE：       最大化对数似然的"下界"（ELBO，见下一章）
GAN：       干脆不算似然，改用对抗损失隐式逼近分布
Diffusion： 把似然拆成一串可计算的去噪步骤
```

## 6. 一个最小例子（估计高斯均值）

```python
import torch

data = torch.tensor([1.0, 2.0, 1.5, 2.5, 2.0])     # 观测数据
mu = torch.zeros(1, requires_grad=True)             # 待估均值

opt = torch.optim.SGD([mu], lr=0.1)
for _ in range(200):
    # 负对数似然（固定方差的高斯），等价于 MSE
    nll = ((data - mu) ** 2).mean()
    opt.zero_grad(); nll.backward(); opt.step()

print(mu.item(), data.mean().item())   # mu 收敛到样本均值 1.8
```

MLE 估计出的高斯均值，正好是样本平均值——直觉与数学一致。

## 本章小结

最大似然估计是“让已观测数据最有可能”，等价于在 KL 散度下让模型分布贴近真实分布。交叉熵、MSE、语言模型损失都是它的特例。生成模型的核心挑战是 `p_θ(x)` 难算，于是 VAE 用 ELBO 下界、GAN 用对抗、Diffusion 用分步去噪来绕开——这是后续各章的统一线索。
