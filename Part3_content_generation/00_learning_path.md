# 00 学习路线（内容生成篇）

## 总体目标

理解主流生成模型（VAE / GAN / Diffusion / Flow Matching）的核心思想与演化关系，能讲清每一种“在优化什么、怎么采样、解决了前一代什么问题”，并落到文生图与视频生成的真实系统。

## 每种模型解决的核心问题

| 模型 | 解决的问题 | 关键思想 |
|---|---|---|
| MLE（根基）| 怎么定义“生成得好” | 让模型分布尽量贴近真实数据分布 |
| VAE | 显式建模分布 + 可采样 | 概率编码器 + 解码器 + 重参数化 |
| GAN | 生成更清晰逼真的样本 | 生成器与判别器对抗 |
| Diffusion | 训练更稳、质量更高 | 加噪 → 学去噪，逐步生成 |
| Score-based | 更统一的数学视角 | 学数据分布的梯度（score）|
| Flow Matching | 统一并简化各类生成 | 学一个把噪声“运输”到数据的向量场 |

## 建议节奏（约 8~10 周）

```text
第1周  AIGC 全景 + 判别 vs 生成 + MLE
第2周  VAE：原理 + 跑通 MNIST/人脸生成
第3周  GAN：对抗训练 + 跑通明星人脸（DCGAN）
第4-5周 Diffusion：前向加噪/反向去噪，DDPM
第6周  Score-based 视角统一理解
第7周  Flow Matching + 动手训练
第8周  文生图：Stable Diffusion 组件拆解
第9-10周 视频生成原理 + Wan + 视频 Agent
```

## 学习原则

- 先抓“一句话直觉”，再看公式。生成模型公式多，但直觉很简单。
- 每种模型都务必跑通一个最小 demo（哪怕只是 MNIST）。
- 重点理解“采样过程”：训练完之后，如何从噪声生成一张图？
- 经典论文读图和摘要即可，先建立全局观。
