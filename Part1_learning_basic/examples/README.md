# examples

本目录用于存放 Part1 的练习代码和 notebook。

建议按以下顺序添加：

1. `01_linear_regression_numpy.ipynb`：NumPy 手写线性回归。
2. `02_sklearn_baselines.ipynb`：线性模型、决策树、随机森林、GBDT 对比。
3. `03_decision_tree_visualization.ipynb`：决策树可视化与剪枝实验。
4. `04_pytorch_mlp.ipynb`：PyTorch MLP 分类。
5. `05_gpu_check.ipynb`：GPU、device、dtype、矩阵乘法速度检查。

原则：

- 每个 notebook 只解决一个主题。
- 每个实验都要有指标和结论。
- 不要只保留成功结果，也要记录踩坑和修复方法。
