# Part1_learning_basic：大模型学习基础篇

本部分不是直接讲大模型，而是补齐理解大模型所需的基础：机器学习、树模型、集成学习、线性模型、深度学习、GPU/CUDA 和 PyTorch。

如果把大模型看成一座高楼，本部分是地基：数据、模型、损失函数、优化、泛化能力、训练流程和计算硬件。

## Part1 学完后应该达到的程度

你应该能做到：

- 能解释什么是监督学习、无监督学习、损失函数、过拟合、欠拟合。
- 能手动说明线性回归、决策树、随机森林、GBDT 的基本原理。
- 能说清楚 CART 如何分裂节点，GBDT 为什么是一棵棵树逐步修正误差。
- 能理解神经网络中的张量、前向传播、反向传播、梯度下降。
- 能看懂向量、矩阵、张量、概率分布、期望、梯度这些基础数学概念在模型训练中的作用。
- 能知道 GPU 为什么适合深度学习，CUDA 在其中扮演什么角色。
- 能用 PyTorch 写出一个标准训练循环。
- 能独立完成一个表格数据建模项目和一个简单神经网络项目。

## 学习建议

- 每个算法都问三个问题：它在优化什么？它怎么做预测？它有什么缺点？
- 不要一开始追求公式严谨性，先用直觉理解，再补数学细节。
- 传统机器学习算法不要跳过。很多大模型工程问题仍然依赖数据划分、指标、过拟合判断和实验设计。
- PyTorch 不要只学 API，要理解 `Tensor`、`autograd`、`nn.Module`、`Dataset/DataLoader` 和训练循环之间的关系。
